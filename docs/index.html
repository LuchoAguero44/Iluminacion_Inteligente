Iluminación Inteligente

Control automático de luces basado en presencia y luz ambiente

Descripción del Proyecto

Este proyecto desarrollado en la materia Lenguaje de Programación Visual implementa un sistema de control de luces basado en sensores de movimiento (PIR) y sensores de luz (LDR). Utilizando LabVIEW y una placa Arduino, el sistema detecta la presencia de personas y la cantidad de luz natural para encender o apagar las luces automáticamente.

Tecnologías y Herramientas

  • 🔧 LabVIEW
  • 💡 Sensor de luz (LDR)
  • 👣 Sensor de movimiento (PIR)
  • 🖥️ Tarjeta controladora (Arduino u otra compatible)
  • 💻 Comunicación serial entre hardware y software

Funcionalidades

  • ✔️ Detección de movimiento para encender/apagar luces.
  • ✔️ Medición de luz ambiente con LDR.
  • ✔️ Umbrales configurables para nivel de iluminación.
  • ✔️ Interfaz gráfica para monitoreo en tiempo real.
  • ✔️ Control automático y/o manual desde la PC.
